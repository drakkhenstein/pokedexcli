# Pokedex Cli

A PokeMon catching game written in go! Explore and catch PokeMon right from the terminal. You can catch all PokeMon, male/female and even shiny, up to Gen 7 and some Gen 8. Use the map commands to find locations and the explore command to find PokeMon at each location. Use the catch command, and the name of the PokeMon, to throw a Pokeball to catch a PokeMon. If the PokeMon is caught you can use the inspect command to see all of its stats.  Once you catch many more PokeMon you can see a list of them all with the pokedex command. Happy shiny hunting!

Prompt Commands:

help - Displays the list of commands in the Pokedex game.

map - Will get you the next page of locations for catching PokeMon.

mapb - Will get you the previous page of locations for catching PokeMon.

explore - Explore a location on the map to find PokeMon.

catch - Throws pokeball in an attempt to catch a PokeMon at current location.

inspect - Inspects caught PokeMon.

pokedex - Lists all of your caught PokeMon.

exit - Exits the Pokedex game.
